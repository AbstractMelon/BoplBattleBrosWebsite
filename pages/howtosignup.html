How to sign up!

Create a User

To create a user, go to the #bots channel in Discord and type the following command:

/createuser

This command will automatically generate a user for you.

Create a Team

To create a team, go to the #bots channel in Discord and type the following command:

/jointournament TeamName

Replace "TeamName" with the desired name for your team. This command will automatically generate a user and associate it with the specified team.
